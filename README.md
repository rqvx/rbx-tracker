### rbx-tracker
Lightweight header-only Roblox update tracker.

Credits to: https://github.com/YuB-W/YuB-X-Hyperion_Updateds

How does it work?
It just fetches DeployHistory.txt from Roblox "setup.rbxcdn.com", simple as that.

See `main.cpp` for usage examples.

> [!WARNING]  
> This is Windows only!
